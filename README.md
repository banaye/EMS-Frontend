-A modern examination system with e-learning platform built using React, TypeScript and Vite
-It's a role-based system for students, Instructors, and Administrators

## Overview
## Features

### For Students
-  Browse and enroll in courses
-  Take exams with timer and auto-submit
-  View exam results and performance analytics
-  Manage personal profile
-  Download course materials
-  Secure exam taking with tab-switch detection

### For Instructors
-  Create and manage courses
-  Create and manage exams
-  Build question banks
-  View student performance
-  Access course analytics and reports
-  Upload course materials

### For Administrators
-  Full user management (students, instructors, admins)
-  Oversee all courses and exams
-  Platform-wide analytics dashboard
-  Generate comprehensive reports
-  System oversight and management

### General Features
-  JWT-based authentication
-  Responsive design for all devices
-  Mobile-friendly interface
-  Real-time notifications
-  Interactive data visualization
-  Auto-save exam answers
-  Timer with auto-submit
-  Exam security measures (tab-switch detection, copy-paste prevention)
